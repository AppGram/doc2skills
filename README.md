# docs2skills

A powerful desktop application for converting documentation into AI-ready skills.

## Installation

Download the latest DMG from the [Releases](../../releases) page.


## Quick Install (Recommended)

```bash
curl -fsSL https://github.com/AppGram/doc2skills/releases/latest/download/install.sh | bash
```

```
brew install --cask appgram/tap/docs2skills
```

This installs the app, dependencies (Playwright), and sets up the CLI.

### Manual macOS Installation

1. Download the `.dmg` file for your architecture (Apple Silicon or Intel)
2. Open the DMG and drag `docs2skills` to Applications
3. On first launch, right-click and select "Open" to bypass Gatekeeper

## Requirements

- macOS 11.0 (Big Sur) or later
- Playwright (for JavaScript-rendered documentation)
